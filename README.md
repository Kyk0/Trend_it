**Personal Project Repository**

Project: Trend It  
Goal: Find similar time-series trends across different metrics/regions within a shared timeframe  
Status: Working locally. GitHub Pages deployment is partially working.  

---

## Tech Stack

### Frontend
- React 18
- React Router DOM
- Tailwind CSS
- Chart.js + react-chartjs-2
- Framer Motion
- Vercel Analytics

### Backend
- Python / Django 4.2
- Django REST Framework
- django-cors-headers
- NumPy, pandas
- SciPy (z-score normalization)
- tslearn (time-series clustering: KMeans, soft-DTW)
- SQLite
- openpyxl

### ML / Data Analysis
- Jupyter Notebook
- tslearn (KMeans, KShape, DTW)
- scikit-learn (silhouette score)
- Matplotlib

### Deployment
- GitHub Pages (frontend, current)
- Render (backend)
- Vercel (frontend, previous)
