# Trend_it

A web application for exploring and visualizing economic trend data using time-series clustering.

## Tech Stack

### Frontend
| Technology | Purpose |
|---|---|
| [React 18](https://react.dev/) | UI framework |
| [React Router DOM](https://reactrouter.com/) | Client-side routing |
| [Tailwind CSS](https://tailwindcss.com/) | Utility-first CSS styling |
| [Chart.js](https://www.chartjs.org/) + [react-chartjs-2](https://react-chartjs-2.js.org/) | Data visualization / charts |
| [Framer Motion](https://www.framer.com/motion/) | Animations |
| [Vercel Analytics](https://vercel.com/analytics) | Usage analytics |

### Backend
| Technology | Purpose |
|---|---|
| [Python](https://www.python.org/) | Server-side language |
| [Django 4.2](https://www.djangoproject.com/) | Web framework |
| [Django REST Framework](https://www.django-rest-framework.org/) | REST API |
| [django-cors-headers](https://github.com/adamchainz/django-cors-headers) | CORS handling |
| [NumPy](https://numpy.org/) | Numerical computing |
| [Pandas](https://pandas.pydata.org/) | Data manipulation |
| [SciPy](https://scipy.org/) | Scientific computing (z-score normalization) |
| [tslearn](https://tslearn.readthedocs.io/) | Time-series clustering (KMeans, soft-DTW) |
| [SQLite](https://www.sqlite.org/) | Embedded database |
| [openpyxl](https://openpyxl.readthedocs.io/) | Excel file parsing |

### ML / Data Analysis
| Technology | Purpose |
|---|---|
| [Jupyter Notebook](https://jupyter.org/) | Exploratory data analysis and model prototyping |
| [Python](https://www.python.org/) | Data processing language |
| [Pandas](https://pandas.pydata.org/) | Data loading and manipulation |
| [NumPy](https://numpy.org/) | Numerical operations |
| [SciPy](https://scipy.org/) | Statistical analysis |
| [tslearn](https://tslearn.readthedocs.io/) | Time-series clustering (KMeans, KShape, DTW) |
| [scikit-learn](https://scikit-learn.org/) | Evaluation metrics (silhouette score) |
| [Matplotlib](https://matplotlib.org/) | Plotting and visualization |
| [SQLite](https://www.sqlite.org/) | Storing processed data |
